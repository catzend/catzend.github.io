CSS的选择器总结
发表于2020-10-09|更新于2020-10-10
|字数总计:1.8k|阅读时长:6分钟|阅读量:

CSS选择器

基本选择器

* 通配符选择器

通用元素选择器，匹配任何元素

Element元素选择器

标签选择器，匹配所有使用E标签的元素

. class类选择器

class选择器，匹配所有class(类名)属性中包含info的元素

# ID选择器

id选择器，匹配所有id属性等于footer的元素

Ele1~Ele2 同级选择器

匹配任何在E元素之后的同级F元素

多元素组合选择器

多元素选择选择器

多元素选择器，同时匹配所有E元素或F元素，E和F之间用逗号分隔

后代选择器

后代元素选择器，匹配所有属于E元素后代的F元素，E和F之间用空格分隔,部分第一代还是第几代

子代选择器

子元素选择器，匹配所有E元素的子元素F,只选择直接的第一代选择器

+相邻选择器

相邻元素选择器，匹配所有紧随E元素之后的

属性选择器

Element[attitude]全部属性选择器

匹配所有具有attitude属性的E元素，不考虑它的值。（注意：E在此处可以省略，比如"[cheacked]"。以下同。）

Element[attr = val]固定属性选择器

匹配所有sttr属性等于"val"的元素

E[attr~=val]其中属性选择器

匹配所有att属性具有多个空格分隔的值、其中一个值等于"val"的E元素

Element[att|=val]开头选择器

匹配所有att属性具有多个连字号分隔（hyphen-separated）的值、其中一个值以"val"开头的E元素，主要用于lang属性，比如"en"、“en-us”、"en-gb"等等

伪类选择器

匹配所有未被点击的链接
匹配所有已被点击的链接
匹配鼠标已经其上按下、还没有释放的E元素
匹配鼠标悬停其上的E元素
匹配获得当前焦点的E元素
匹配lang属性等于c的E元素

Ele:first-child第一子元素选择器

匹配父元素的第一个子元素

链接选择器

E:link匹配所有未被点击的链接
E:visited匹配所有已被点击的链接
E:active匹配鼠标已经其上按下、还没有释放的元素
E:hover匹配鼠标悬停其上的E元素

Element:focus焦点选择器

匹配获得当前焦点的元素

Element:lang©

q:lang(sv) { quotes: “\201D” “\201D” “\2019” “\2019”; }

Element:enabled

匹配表单中激活的元素

Element:disabled

匹配表单中禁用的元素

Element:checked

匹配表单中被选中的单选框或者多选框

Element::selection

匹配用户当前选中的元素

结构性伪类元素

Element:root

匹配文档的根元素，对于HTML文档，就是HTML元素

Element:empty

:empty选择器选择每个没有任何子级的元素（包括文本节点）。

Element:last-child

:last-child选择器用来匹配父元素中最后一个子元素。

Element:nth-child(n)

匹配其父元素的第n个子元素，第一个编号为1

:nth-child(n) 选择器匹配父元素中的第 n 个子元素，元素类型没有限制。

n 可以是一个数字，一个关键字，或者一个公式。

Element:nth-last-child(n)

匹配其父元素的倒数第n个子元素，第一个编号为1

:nth-last-child(n) 选择器匹配属于其元素的第 N 个子元素的每个元素，不论元素的类型，从最后一个子元素开始计数。

n可以是一个数字，一个关键字，或者一个公式。

Element:nth-of-type(n)

与:nth-child()作用类似，但是仅匹配使用同种标签的元素

：nth-of-type（n）选择器匹配同类型中的第n个同级兄弟元素。

n可以是一个数字，一个关键字，或者一个公式。

Element:nth-last-of-type(n)

:nth-last-of-type(n)选择器匹配同类型中的倒数第n个同级兄弟元素。

n 可以是一个数字，一个关键字，或者一个公式。

Element:first-of-type

:first-of-type 选择器匹配元素其父级是特定类型的第一个子元素。

Element:last-of-type

:last-of-type选择器匹配元素其父级是特定类型的最后一个子元素。

Element:only-child

:only-child 选择器匹配属于父元素中唯一子元素的元素。

Element:only-of-type

:only-of-type 代表了任意一个元素，这个元素没有其他相同类型的兄弟元素。

CSS 3的反选伪类

:not(selector) 选择器匹配每个元素是不是指定的元素/选择器。

CSS 3中的 :target 伪类

＃ 锚的名称是在一个文件中链接到某个元素的URL。元素被链接到目标元素。

:target选择器可用于当前活动的target元素的样式。

伪元素

Element:first-lin首行选择器

匹配E元素的第一行

Element:first-letter

匹配E元素的第一个字母

Element:before

在E元素之前插入生成的内容

Element:after

在E元素之后插入生成的内容
