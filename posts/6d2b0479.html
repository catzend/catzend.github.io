Javascript语法

区分大小写

ECMAScript中的一切（变量、函数名和操作符）都区分大小写。变景名test和变城名Test分别表示两个不同的变量，而函数名不能使用typeof,因为它 是一个关键字，但typeOf则完全可以是-个有效的函数名。

标识符

标识符，就是指变量、函数、属性的名字，或者函数的参数。

(经常有人问标识符,你就可以直接理解为一个标识,无论是函数名还是参数或者变量等)

  1. 第一个字符必须是一个字母、下划线（_）或一个美元符号（$）;
  2. 其他字符可以是字母、下划线、美元符号或数字。

严格模式

在脚本顶部添加"use strict"，其实是一个编译指示，用于告诉JavaScript引擎切换到严格模式。

在严格模式下，ECMAScript3中的一些不确定的行为将得到处理，而且对某些不安全 的操作也会抛出错误。

语句

  1. 分号结尾！(一定养成习惯)
  2. 对于控制语句，推荐用{ }代码块，让编码易读，且意图清晰！

ECMAScript的变量是可以用来保存任何类型的数据。

      • 如果字符串中包含有效的浮点格式，如则将其转换为对应的浮点数值(同样，也会忽 略前导零);
      • 如果字符串中包含有效的十六进制格式，例如"Oxf",则将其转换为相同大小的十进制整 数值；
      • 如果字符串是空的(不包含任何字符)，则将其转换为0;
      • 如果字符串中包含除上述格式之外的字符，则将其转换为NaN。

String 类型
  1. String类型用于表示由零或多个16位Unicode字符组成的字符序列,即字符串.
  2. 字符串可以由单引号或者双引号表示,不能是一个单一个双.
  3. 以双引号开头的字符串 也必须以双引号结尾，而以单引号开头的字符串必须以单引号结尾

  • 字符串的特点

    • ECMAScript中的字符串是不可变的,也就是字符串一旦建立,他们的值就不能改变

    • 如果想改变某个变量保存的字符串,首先要销毁原来的字符串,然后再用另一个包含数值的字符串填充该变量

      var lang = “Java”

      lang = lang + “Script”

      变量lang开始时包含字符申"Java",而第二行代码把lang的值重新定义为"Java” 与"Script"的组合，即"JavaScript"实现这个操作的过程如下：首先创建一个能容纳10个字符的新字符串.然后在这个字符串中填充"Java"和"Script",最后一步是销毁原來的字符串"Java"和字符串"Script",因为这两个字符串已经没用了。这个过程是在后台发生的，而这也是在某些旧版木的 浏览器(例如版本低于1.0的Firefbx、IE6等)中拼接字符串时速度很慢的原因所在。但这些浏览器后 来的版本已经解决了这个低效率问题。

Object 类型

  1. ECMAScript中的对象其实就是一组数据和功能的集合。

  2. 对象可以通过执行new操作符后跟要创建的对象类型的名称来创建

  3. 创建的Object类型的实例并为其添加属性和方法,就可以创建自定义对象

    var o = new Object();

    如果不给构造函数传递参数,则可以省略后面的那一对圆括号

    var o = new Object; // 有效但是不推荐

  • Object的每个实例都具有下列属性和方法:
    • constructor:保存着用于创建当前对象的函数,对于前面的列子而言,构造函数(constructor)就是Object();
    • hasOwnProperty (propertyName):用于检查给定的属性在当前对象实例中(而不是在实例的原型中)是否存在。其中，作为参数的属性名**(propertyName**〉必须以字符串形式指定(例 如：o.hasOwnProperty (“name”) ).
    • isProtocypeOf (object)：用于检査传入的对象是否是另一个对象的原型。
    • propertylsEnumerable(propertyName):用于检查给定的属性是否能够使用for-in语句 (本章后面将会讨论)来枚举。与hasOwnProperty ()方法–样，作为参数的域性名必须以字符 串形式指定。
    • toLocaleString()：返回对象的字符串表示，该字符串与执行环境的地区对应。
    • toString() :返回对象的字符串表示。
    • valueOf():返回对象的字符串、数值或布尔值表示。通常与toStringO方法的返面值 相同。

一元操作符

只能操作一个值的操作符叫做一元操作符。

  1. 递增递减操作符(++/–)

    1.var age = 10;

    sum = age++;

    console.log(sum) === 10

    console.log(age) === 11

    num++/-- =====>> 就是先使用num的值,然后进行num+1或者num-1

    ++num/–num ====>> 先进行num+1/num-1,然后进行使用加减后的值

    总结
    前置++：先加1，后参与运算
    后置++：先参与运算，后加1
    上面两个理解后，下面两个自通
    前置-- ：先减1，后参与运算
    后置-- ：先参与运算，后减
